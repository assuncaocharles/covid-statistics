# Covid Statistics (Covid Stats)

A simple project to display clean and concise Covid-19 Statistics by country.

Covid-19 Data Source: https://github.com/owid/covid-19-data

## Stack

- ⚛️ React.js
- ⚛️ React Query
- 💅 Tailwind CSS
- 🔎 Google Analytics
- 🔂 CI/CD & Deploy
